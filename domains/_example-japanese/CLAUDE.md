# CLAUDE.md — _example-japanese

This is an example domain used to demonstrate the workspace conventions. Do **not**
treat it as an active learning target unless the user explicitly says so.

## Goal

Reach conversational Japanese (roughly N5→N4) with strong grammar intuition, not
textbook memorization.

## Starting level

Complete beginner. Knows that hiragana exists and can recognize a few characters
(`あ`, `い`, `う`) from anime context. No grammar, no vocabulary.

## Methodology

- Grammar: Cure Dolly / guidetojapanese.org approach — explain the structural logic
  (invisible ゼロ subject, particle roles) before patterns.
- Writing system: hiragana first, then katakana. Kanji is **out of scope** (handled
  externally via WaniKani).
- Vocabulary: high-frequency first, via `card` blocks + SRS.
- Drill: short, frequent, focused on one pattern at a time.

## Preferred sources / authors

- Tae Kim — *Guide to Japanese Grammar* (guidetojapanese.org).
- Cure Dolly — *Organic Japanese* video series (YouTube).

## Explanation style

- Always surface the "hidden ゼロ subject" when analyzing sentences.
- Give Russian gloss first, then a literal structural breakdown.
- Prefer visual/spatial analogies over tables.
- Show kana with romaji only in the first introduction of a new character; drop
  romaji afterwards.

## Out of scope

- Kanji learning (WaniKani).
- JLPT test prep specifics.
- Keigo / business Japanese.
