# Урок 02 — Ряд か

Второй ряд годзюона: пять слогов с согласным `k`. После этого ряда уже можно читать
десятки реальных слов.

## Разогрев

Повтори гласные из [`01-intro`](./01-intro.md) — `あ い う え お`. Если какая-то
карточка ещё `learning`, прогоняй SRS до `known`, и только потом переходи к ряду `か`.

## Новая тема

| символ | звук | образ-подсказка                             |
|--------|------|---------------------------------------------|
| `か`   | ka   | «крестик с хвостиком»                       |
| `き`   | ki   | «ключ» — две горизонтали + крюк             |
| `く`   | ku   | «клюв» — угол, открытый вправо              |
| `け`   | ke   | вертикаль + «く» справа                     |
| `こ`   | ko   | две короткие горизонтали                    |

Произношение согласного `k` — как русское «к», но без придыхания. Гласные — те же,
что в ряду `あ`: `か` = `k` + `а`, `き` = `k` + `и` и так далее.

## Разобранные примеры

- `かお` → *kao* → «лицо».
- `いけ` → *ike* → «пруд».
- `きく` → *kiku* → «слушать» (глагол в словарной форме).
- `こえ` → *koe* → «голос».
- `あき` → *aki* → «осень».

## Итог

Пять слогов ряда `か` — `か き く け こ`. Связка с гласными работает прямолинейно:
берём согласный `k`, подставляем любую из пяти гласных. Теперь уже можно составлять
слова: `かお`, `いけ`, `あき`, `こえ`. Следующий шаг — ряд `さ` (не входит в пример).
