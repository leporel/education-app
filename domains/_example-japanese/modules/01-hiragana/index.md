# Хирагана

## Цель

- Уметь читать любой текст на хирагане без запинки.
- Писать все 46 символов базового ряда от руки с правильным порядком черт.

## Предпосылки

Нет. Это первый модуль домена.

## Структура

- [`theory.md`](./theory.md) — что такое хирагана, структура таблицы, произношение.
- [`cards.md`](./cards.md) — карточки по всем рядам.
- [`drills.md`](./drills.md) — упражнения на чтение и различение.
- [`lessons/`](./lessons/) — уроки по рядам.

## Порядок прохождения

1. Прочитать [`theory.md`](./theory.md).
2. Пройти [`lessons/01-intro.md`](./lessons/01-intro.md) — ряд `あ`.
3. Пройти [`lessons/02-row-ka.md`](./lessons/02-row-ka.md) — ряд `か`.
4. Прогонять SRS по ряду до состояния `known`, затем — следующий ряд.
5. Drill'ы из [`drills.md`](./drills.md) — как смешанная практика между уроками.
