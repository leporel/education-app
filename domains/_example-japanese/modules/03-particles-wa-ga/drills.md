# Упражнения — は и が

## Выбор частицы (multiple-choice)

```drill
type: multiple-choice
prompt: "Вставь частицу: わたし ___ 学生 です."
options: [は, が, を, に]
answer: は
check: exact
hint: введение темы «что касается меня…»
```

```drill
type: multiple-choice
prompt: "Вставь частицу: だれ ___ きた？"
options: [は, が, も, と]
answer: が
check: exact
hint: вопросительное слово никогда не бывает темой
```

```drill
type: multiple-choice
prompt: "Контраст двух тем: わたし ___ いく、かれ ___ いかない."
options: ["は / は", "が / が", "は / が", "が / は"]
answer: "は / は"
check: exact
```

## Fill-in (hiragana)

```drill
type: fill-in
prompt: "Вставь ОДНУ частицу в клетку: ねこ ___ いる."
answer: が
check: exact
hint: констатация наличия — новая информация
```

## Свободный ответ с fuzzy

```drill
type: free-form
prompt: "Объясни в одной фразе: почему в «りんご は 赤い» частица は, а не が?"
answer: "потому что りんご — это тема (о чём речь), а не субъект; субъект — скрытое ゼロ"
check: fuzzy
hint: ключевые слова — «тема», «ゼロ», «субъект»
```

## Разбор с ручной проверкой

```drill
type: free-form
prompt: "Разбери структуру: «きょう は いい てんき です». Где тема? Где ゼロ-субъект? Что сказуемое?"
answer: "Тема: きょう (сегодня). ゼロ-субъект: [сегодняшняя погода]-が. Сказуемое: いい てんき です."
check: manual
```
