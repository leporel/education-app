# Japanese — пример-домен

Полноценный пример-домен, демонстрирующий все фичи воркспейса: frontmatter, карточки
(`card`-блоки), все четыре типа drill'ов (translate / fill-in / multiple-choice /
free-form) и все режимы проверки (exact / fuzzy / regex / manual), статусы прогресса,
roadmap и память.

## Файлы

- [`CLAUDE.md`](./CLAUDE.md) — инструкции для Claude по этому домену.
- [`memory.md`](./memory.md) — память домена.
- [`roadmap.md`](./roadmap.md) — план обучения.

## Материалы и заметки

- [`misc/`](./misc/) — справочные ассеты (например, `hiragana-chart.svg` и
  подпапка `Images/`). В UI доступны через вкладку «Материалы» с превью картинок.
- [`notes/`](./notes/) — личные заметки пользователя (две штуки: прогресс и
  частые ошибки). Управляются через правый сайдбар в UI.

## Модули

- [`01-hiragana`](./modules/01-hiragana/index.md) — азбука хирагана: теория, карточки,
  упражнения, два урока.
- [`02-katakana`](./modules/02-katakana/index.md) — катакана: теория + карточки +
  первый урок.
- [`03-particles-wa-ga`](./modules/03-particles-wa-ga/index.md) — грамматика: частицы
  は и が через концепцию ゼロ-субъекта.
