# Карточки

Каждая карточка — fenced-блок с языком `card`. Поля `front` и `back` обязательны,
`hint` и `tags` — опциональны. Блоки можно группировать под подзаголовками.

## <Группа 1>

```card
front: <вопрос / слово / стимул>
back: <ответ>
hint: <необязательная подсказка>
tags: [<tag>, ...]
```

```card
front: ...
back: ...
```

## <Группа 2>

```card
front: ...
back: ...
```
